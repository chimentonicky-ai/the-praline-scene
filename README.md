# The Praline Scene - Premium New Orleans Praline E-Commerce

A luxury e-commerce website for The Praline Scene, a family-owned New Orleans praline company.

## Brand Overview

- **Authentic New Orleans heritage** with family-owned tradition
- **Handmade in small batches** using a treasured family recipe
- **Premium gourmet confectionery** with luxury bakery aesthetic
- **Color palette:** Soft blush pink, cream, caramel, chocolate brown, gold accents
